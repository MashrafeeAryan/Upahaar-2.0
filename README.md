# Upahaar

**Remember the little things. Show you care.**

Upahaar is a thoughtful gifting app that helps people remember small details about their friends and turn those details into better gift ideas.

Instead of starting with a product catalog, Upahaar starts with the person. Users can save birthdays, likes, dislikes, memories, gift clues, and little things someone casually mentions. Later, Mimi, the app’s gift helper, can use those saved details to suggest more personal gifts.

---

## Why I Built This

I wanted to build something around a simple but real problem: meaningful gifts are usually not about buying the most expensive thing. They are about remembering.

Someone might casually say:

> “I really want a better coffee mug.”

or

> “I love gold jewelry, but only simple designs.”

At the time, it feels easy to remember. But when their birthday comes around months later, those small details are often forgotten.

Upahaar is built to help with that.

The goal is to make it easy to save those little details in the moment, so later you can give a gift that feels personal and thoughtful.

---

## Features

### Home Dashboard

The home screen gives users a quick view of:

- The next upcoming birthday
- Recent saved memories
- Quick actions like Memory Drop and Add Friend
- A floating Mimi assistant button

This makes the app feel simple and easy to use from the start.

---

### Friends

Users can save people they care about, including:

- Name
- Birthday
- Relationship
- Likes
- Gift clues
- Saved memories

Each friend card shows useful details, such as birthday countdowns and example saved notes.

---

### Memory Drop

Memory Drop is one of the main ideas behind Upahaar.

It lets users quickly save something someone mentioned in a conversation.

Example:

> “Aisha said she loves matcha and simple gold jewelry.”

That memory can later help generate better gift ideas.

The idea is:

> Hear something. Save it quickly. Use it later when it matters.

---

### Mimi Gift Helper

Mimi is the app’s gift helper.

Mimi uses saved memories and preferences to suggest thoughtful gift ideas. Instead of giving random gifts, Mimi explains why each gift fits the person.

Example:

- Gift idea: Minimal coffee mug
- Why it fits: They mentioned wanting a better mug
- Make it personal: Add their name or a small custom design

---

### Gift Ideas and Customization

Users can explore gift suggestions and customize ideas with:

- Product type
- Color
- Style
- Personal note
- Custom message

This helps complete the flow from remembering a detail to preparing a thoughtful gift.

---

### Widget Preview Concept

Upahaar also includes a widget preview concept for quick memory capture.

The idea is that if a friend says something useful during a conversation, the user could quickly open the Memory Drop widget and type a short note before forgetting it.

Example:

> “Rahim said he wants a simple black hoodie.”

Later, Mimi can use that detail to suggest a better gift.

This is meant to support the core idea of Upahaar:

> No small details missed.

## The Comeback Story

Before this challenge, Upahaar was an unfinished gifting app idea. It had the basic direction of helping people with gifts, but it did not yet feel clear or meaningful.

During Finish-Up-A-Thon, I changed the direction of the project.

Instead of building just another gift app, I focused on the question:

> What actually makes a gift thoughtful?

The answer was memory.

So I rebuilt the project around remembering people better. I added Memory Drop, friend profiles, Mimi’s gift helper, gift customization, and a widget-ready quick capture idea.

Before:

> A basic unfinished gifting app idea.

After:

> A thoughtful gifting assistant that helps users save memories, remember birthdays, and turn small details into personal gift ideas.
